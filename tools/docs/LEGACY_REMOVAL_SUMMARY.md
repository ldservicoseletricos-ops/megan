# Legacy Removal Summary

Esta versão limpa removeu da raiz oficial referências legadas e paralelas, priorizando apenas a base principal:

- frontend oficial: `frontend/`
- backend oficial: `backend/`

Pastas legadas removidas da raiz oficial nesta versão limpa incluíam variantes antigas de frontend, backend e módulos de apoio experimentais.

Esta limpeza foi feita para consolidar a arquitetura e reduzir confusão operacional.
