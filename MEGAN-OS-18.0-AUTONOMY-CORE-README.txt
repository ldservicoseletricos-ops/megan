MEGAN OS 18.0 AUTONOMY CORE

Esta versão adiciona autonomia supervisionada real ao projeto Megan OS.

Principais recursos:
- Novo painel Autonomy Core 18.0 no frontend.
- Nova API /api/autonomy-core-18 no backend.
- Ciclo de análise operacional.
- Criação automática de tarefas pendentes de validação humana.
- Relatório executivo da saúde do sistema.
- Chat diretor de autonomia conectado ao backend.
- Checklist vivo de backend, frontend, memória e estrutura.

Importante:
A Megan OS 18.0 NÃO executa ações sensíveis sozinha. Ela observa, analisa, cria tarefas e aguarda aprovação humana.

Como rodar:
1. Backend:
   cd backend
   npm install
   npm start

2. Frontend:
   cd frontend
   npm install
   npm run dev

Endpoints novos:
GET  /api/autonomy-core-18/status
POST /api/autonomy-core-18/cycle
POST /api/autonomy-core-18/report
GET  /api/autonomy-core-18/tasks
POST /api/autonomy-core-18/tasks/:id/status
POST /api/autonomy-core-18/chat
